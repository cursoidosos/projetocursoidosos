Curso de Introdução às Tecnologias para Idosos


O foco do curso é oportunizar o aprendizado de idosos para uso de tecnologias, incentivando-os a iniciar um novo aprendizado e continuar o aprendendo a construir novos conhecimentos necessários para o século XXI.
